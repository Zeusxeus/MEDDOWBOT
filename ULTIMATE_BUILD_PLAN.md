## PART 1: SYSTEM MENTAL MODEL

Before writing a single line, understand what this system is.

**It is NOT a bot that downloads files.**  
**It IS a durable, distributed media processing pipeline with a Telegram interface.**

The distinction matters for every design decision:

| If you think "bot that downloads" | If you think "pipeline with Telegram UI" |
|---|---|
| You put download logic in handlers | Handlers only enqueue work |
| You use asyncio.Queue | You use Redis-backed persistent queue |
| App restart loses all jobs | Jobs survive anything |
| One crash = user timeout | One crash = automatic retry |
| Hard to scale | `--scale worker=5` |

Keep this framing in mind throughout.

## PART 4: CONFIGURATION SYSTEM

**Rule:** Nothing reads `os.environ` directly. Everything goes through `config/settings.py`.  
Missing required vars crash at startup with a clear message — never at user request time.

```python
from __future__ import annotations

from pathlib import Path
from typing import Optional

from pydantic import Field, field_validator, model_validator
from pydantic_settings import BaseSettings, SettingsConfigDict


class BotSettings(BaseSettings):
    token: str = Field(..., description="Telegram bot token from @BotFather")
    webhook_url: str = Field(default="", description="Full HTTPS webhook URL (prod only)")
    webhook_secret: str = Field(default="", description="Telegram webhook secret token")
    admin_ids: list[int] = Field(default_factory=list, description="Telegram user IDs with admin access")
    
    @field_validator("admin_ids", mode="before")
    @classmethod
    def parse_admin_ids(cls, v: str | list) -> list[int]:
        if isinstance(v, str):
            return [int(x.strip()) for x in v.split(",") if x.strip()]
        return v


class LocalApiSettings(BaseSettings):
    enabled: bool = Field(default=False, description="Use local Bot API server (removes 50MB limit)")
    url: str = Field(default="http://telegram-bot-api:8081", description="Local API server base URL")
    api_id: str = Field(default="", description="Telegram API ID (required for local API)")
    api_hash: str = Field(default="", description="Telegram API hash (required for local API)")
    
    @model_validator(mode="after")
    def validate_local_api(self) -> "LocalApiSettings":
        if self.enabled and (not self.api_id or not self.api_hash):
            raise ValueError("LOCAL_API_ID and LOCAL_API_HASH required when LOCAL_API_ENABLED=true")
        return self


class DatabaseSettings(BaseSettings):
    url: str = Field(
        default="sqlite+aiosqlite:///./dev.db",
        description="SQLAlchemy async DB URL"
    )
    pool_size: int = Field(default=10, ge=1)
    max_overflow: int = Field(default=20, ge=0)
    echo: bool = Field(default=False, description="Log all SQL (dev only)")


class RedisSettings(BaseSettings):
    url: str = Field(default="redis://localhost:6379/0")
    max_connections: int = Field(default=50, ge=1)
    # Database slots
    # DB 0: Taskiq job queue
    # DB 1: Taskiq result backend
    # DB 2: Rate limiting, FSM state, preflight cache, progress pub/sub
    queue_db: int = Field(default=0)
    result_db: int = Field(default=1)
    cache_db: int = Field(default=2)


class WorkerSettings(BaseSettings):
    concurrency: int = Field(default=3, ge=1, description="Tasks per worker process")
    max_retries: int = Field(default=3, ge=0)
    retry_delay: int = Field(default=30, ge=5, description="Seconds between retries")
    job_timeout: int = Field(default=600, ge=60, description="Hard deadline per job (seconds)")
    preflight_timeout: int = Field(default=30, ge=10, description="Metadata fetch timeout")


class RateLimitSettings(BaseSettings):
    per_hour: int = Field(default=10, ge=1, description="Max downloads per user per hour")
    burst: int = Field(default=3, ge=1, description="Short burst allowance above sustained rate")
    max_concurrent_jobs: int = Field(default=2, ge=1, description="Max simultaneous jobs per user")


class DiskSettings(BaseSettings):
    tmp_dir: Path = Field(default=Path("/tmp/bot"))
    min_free_gb: float = Field(default=2.0, ge=0.5, description="Min free disk before refusing jobs")


class FFmpegSettings(BaseSettings):
    target_mb: int = Field(default=45, ge=10, description="Target compressed size")
    max_size_mb: int = Field(default=49, ge=20, description="Upload size limit (standard API)")
    large_file_warn_mb: int = Field(default=200, ge=50, description="Warn before downloading files over this size")
    hw_accel: str = Field(default="auto", description="auto | nvenc | vaapi | none")
    # auto: detect at startup, use if available, fall back to libx264
    # nvenc: force NVIDIA GPU encode
    # vaapi: force Intel/AMD GPU encode  
    # none: always use libx264 (CPU)


class ProxySettings(BaseSettings):
    """
    Proxy configuration for yt-dlp.
    
    Proxy string format (from your list):
        host:port:username:password
        e.g. 45.56.180.147:8381:jhrwjggg:axrxuy5m3k18
    
    yt-dlp expects:
        http://username:password@host:port
    
    We store in DB and convert at use time.
    """
    enabled: bool = Field(default=False, description="Enable proxy pool for downloads")
    # Platforms that ALWAYS use proxy (never bypass)
    force_proxy_platforms: list[str] = Field(
        default_factory=lambda: ["youtube.com", "youtu.be"],
        description="Platforms that always route through proxy"
    )
    # Platforms that NEVER use proxy
    no_proxy_platforms: list[str] = Field(
        default_factory=list,
        description="Platforms that bypass proxy even when pool is enabled"
    )
    health_check_interval: int = Field(default=300, ge=60, description="Seconds between proxy health checks")
    health_check_url: str = Field(
        default="https://www.youtube.com/generate_204",
        description="URL used to verify proxy connectivity"
    )
    health_check_timeout: int = Field(default=10, ge=3, description="Seconds before proxy health check fails")
    rotation_strategy: str = Field(
        default="round_robin",
        description="round_robin | random | least_used | least_errors"
    )
    max_consecutive_failures: int = Field(default=3, ge=1, description="Failures before marking proxy dead")
    dead_proxy_cooldown: int = Field(default=600, ge=60, description="Seconds before retrying dead proxy")


class CookieSettings(BaseSettings):
    """
    YouTube cookie configuration.
    
    Cookies allow downloading:
    - Age-restricted videos
    - Members-only content
    - Private videos you have access to
    - Content requiring login
    
    Cookie file format: Netscape HTTP Cookie File (exported by browser extension)
    Recommended export tool: "Get cookies.txt LOCALLY" Chrome extension
    """
    enabled: bool = Field(default=False, description="Enable cookie authentication for YouTube")
    cookies_dir: Path = Field(default=Path("/app/data/cookies"), description="Directory for cookie files")
    youtube_cookie_file: str = Field(default="youtube.txt", description="YouTube cookie filename")
    # Cookie validation: check that cookies are not expired
    validate_on_startup: bool = Field(default=True, description="Verify cookies work at startup")
    # Platforms that use cookies
    cookie_platforms: list[str] = Field(
        default_factory=lambda: ["youtube.com", "youtu.be"],
        description="Platforms that use cookie auth"
    )


class ObservabilitySettings(BaseSettings):
    log_level: str = Field(default="INFO")
    metrics_port: int = Field(default=9090)


class Settings(BaseSettings):
    model_config = SettingsConfigDict(
        env_file=".env",
        env_file_encoding="utf-8",
        env_nested_delimiter="__",  # BOT__TOKEN, REDIS__URL, etc.
        case_sensitive=False,
        extra="ignore",
    )
    
    env: str = Field(default="development", description="development | production")
    
    bot: BotSettings = Field(default_factory=BotSettings)
    local_api: LocalApiSettings = Field(default_factory=LocalApiSettings)
    database: DatabaseSettings = Field(default_factory=DatabaseSettings)
    redis: RedisSettings = Field(default_factory=RedisSettings)
    worker: WorkerSettings = Field(default_factory=WorkerSettings)
    rate_limit: RateLimitSettings = Field(default_factory=RateLimitSettings)
    disk: DiskSettings = Field(default_factory=DiskSettings)
    ffmpeg: FFmpegSettings = Field(default_factory=FFmpegSettings)
    proxy: ProxySettings = Field(default_factory=ProxySettings)
    cookies: CookieSettings = Field(default_factory=CookieSettings)
    obs: ObservabilitySettings = Field(default_factory=ObservabilitySettings)
    
    @property
    def is_production(self) -> bool:
        return self.env == "production"
    
    @property
    def is_development(self) -> bool:
        return self.env == "development"


# Module-level singleton — import this everywhere
settings = Settings()
```

### database/models.py

```python
from __future__ import annotations

import enum
import uuid
from datetime import datetime
from typing import Optional

from sqlalchemy import (
    BigInteger, Boolean, DateTime, Enum, ForeignKey,
    Index, Integer, String, Text, UniqueConstraint,
    func,
)
from sqlalchemy.dialects.postgresql import UUID as PG_UUID
from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column, relationship


class Base(DeclarativeBase):
    pass


# ─────────────────────────────────────────────
# ENUMS
# ─────────────────────────────────────────────

class JobStatus(str, enum.Enum):
    PENDING = "pending"       # Enqueued, not started
    RUNNING = "running"       # Worker claimed it
    DONE = "done"             # Uploaded to Telegram
    FAILED = "failed"         # Exhausted all retries
    CANCELLED = "cancelled"   # User cancelled via /cancel


class ProxyStatus(str, enum.Enum):
    ACTIVE = "active"         # Passing health checks, in rotation
    DEAD = "dead"             # Failed max_consecutive_failures checks
    DISABLED = "disabled"     # Admin manually disabled
    TESTING = "testing"       # Currently being health-checked


class ProxyRotationStrategy(str, enum.Enum):
    ROUND_ROBIN = "round_robin"
    RANDOM = "random"
    LEAST_USED = "least_used"
    LEAST_ERRORS = "least_errors"


# ─────────────────────────────────────────────
# USER
# ─────────────────────────────────────────────

class User(Base):
    """
    Every Telegram user who interacts with the bot.
    Created on first message via AuthMiddleware.
    """
    __tablename__ = "users"

    id: Mapped[uuid.UUID] = mapped_column(
        PG_UUID(as_uuid=True), primary_key=True, default=uuid.uuid4
    )
    telegram_id: Mapped[int] = mapped_column(
        BigInteger, unique=True, index=True, nullable=False
    )
    # telegram_id is unique per user across Telegram, never changes
    
    username: Mapped[Optional[str]] = mapped_column(String(255), nullable=True)
    # Can be None if user has no @username
    
    first_name: Mapped[Optional[str]] = mapped_column(String(255), nullable=True)
    is_banned: Mapped[bool] = mapped_column(Boolean, default=False, nullable=False)
    is_admin: Mapped[bool] = mapped_column(Boolean, default=False, nullable=False)
    # is_admin = True for users in BOT__ADMIN_IDS OR promoted via /admin promote
    
    rate_limit_override: Mapped[Optional[int]] = mapped_column(
        Integer, nullable=True
    )
    # None = use global setting. Integer = override for this user.
    # Admin can set this to 0 (ban from rate limit perspective) or 100 (VIP)
    
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now(), nullable=False
    )
    last_seen_at: Mapped[Optional[datetime]] = mapped_column(
        DateTime(timezone=True), nullable=True
    )
    # Updated by AuthMiddleware on every message
    
    total_downloads: Mapped[int] = mapped_column(Integer, default=0, nullable=False)
    total_bytes_served: Mapped[int] = mapped_column(BigInteger, default=0, nullable=False)
    # Denormalized counters — updated when jobs complete. Avoids COUNT() queries for /stats

    # Relationships
    settings: Mapped[Optional["UserSettings"]] = relationship(
        back_populates="user", uselist=False, lazy="select"
    )
    jobs: Mapped[list["DownloadJob"]] = relationship(
        back_populates="user", lazy="select"
    )

    def __repr__(self) -> str:
        return f"<User telegram_id={self.telegram_id} username={self.username}>"


# ─────────────────────────────────────────────
# USER SETTINGS
# ─────────────────────────────────────────────

class UserSettings(Base):
    """
    Per-user preferences. One row per user (1:1 with User).
    Created with defaults on first /settings access or first download.
    """
    __tablename__ = "user_settings"

    user_id: Mapped[uuid.UUID] = mapped_column(
        PG_UUID(as_uuid=True),
        ForeignKey("users.id", ondelete="CASCADE"),
        primary_key=True,
    )
    # Primary key = FK. Enforces 1:1 relationship at DB level.

    # Video quality preference
    # "best" = highest quality yt-dlp selects
    # "1080", "720", "480", "360" = max resolution
    # "audio" = audio-only (mp3/m4a)
    format_quality: Mapped[str] = mapped_column(
        String(10), default="720", nullable=False
    )
    
    # Whether to compress videos that exceed 49MB
    # If False AND Local Bot API disabled: large files will be sent as document if possible
    compression_enabled: Mapped[bool] = mapped_column(Boolean, default=True, nullable=False)
    
    # Bundle multiple files into a .zip (for Reddit bulk downloads)
    zip_files: Mapped[bool] = mapped_column(Boolean, default=False, nullable=False)
    
    # Show progress updates while downloading
    # False = only notify on completion
    show_progress: Mapped[bool] = mapped_column(Boolean, default=True, nullable=False)

class Proxy(Base):
    """
    Residential proxy pool for yt-dlp routing.
    
    Input format from admin: host:port:username:password
    yt-dlp format: http://username:password@host:port
    
    Proxies are:
    1. Added via /admin proxy add <host:port:user:pass>
    2. Health-checked every PROXY__HEALTH_CHECK_INTERVAL seconds
    3. Auto-marked DEAD after MAX_CONSECUTIVE_FAILURES failed checks
    4. Auto-recovered after DEAD_PROXY_COOLDOWN seconds
    5. Rotated according to PROXY__ROTATION_STRATEGY
    """
    __tablename__ = "proxies"

    id: Mapped[uuid.UUID] = mapped_column(
        PG_UUID(as_uuid=True), primary_key=True, default=uuid.uuid4
    )
    
    # ── Connection details ──
    host: Mapped[str] = mapped_column(String(255), nullable=False)
    port: Mapped[int] = mapped_column(Integer, nullable=False)
    username: Mapped[str] = mapped_column(String(255), nullable=False)
    password: Mapped[str] = mapped_column(String(255), nullable=False)
    # Note: Password stored plaintext in DB. Use DB encryption at rest in high-security envs.
    
    label: Mapped[Optional[str]] = mapped_column(String(100), nullable=True)
    # Optional human-readable label, e.g. "US-West-1", "residential-pool-A"
    
    # ── Status ──
    status: Mapped[ProxyStatus] = mapped_column(
        Enum(ProxyStatus), default=ProxyStatus.ACTIVE, nullable=False, index=True
    )
    
    # ── Health tracking ──
    last_checked_at: Mapped[Optional[datetime]] = mapped_column(
        DateTime(timezone=True), nullable=True
    )
    last_success_at: Mapped[Optional[datetime]] = mapped_column(
        DateTime(timezone=True), nullable=True
    )
    consecutive_failures: Mapped[int] = mapped_column(Integer, default=0, nullable=False)
    total_failures: Mapped[int] = mapped_column(Integer, default=0, nullable=False)
    total_successes: Mapped[int] = mapped_column(Integer, default=0, nullable=False)
    avg_latency_ms: Mapped[Optional[float]] = mapped_column(nullable=True)
    # Running average latency from health checks
    
    # ── Usage tracking (for rotation strategies) ──
    total_uses: Mapped[int] = mapped_column(Integer, default=0, nullable=False)
    last_used_at: Mapped[Optional[datetime]] = mapped_column(
        DateTime(timezone=True), nullable=True
    )
    # Used for round_robin (order by last_used_at ASC) and least_used (order by total_uses ASC)
    
    # ── Rotation ──
    # For round_robin: worker picks proxy where last_used_at is oldest
    # This auto-distributes load without a shared counter
    
    added_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True), server_default=func.now(), nullable=False
    )
    added_by_user_id: Mapped[Optional[uuid.UUID]] = mapped_column(
        PG_UUID(as_uuid=True),
        ForeignKey("users.id", ondelete="SET NULL"),
        nullable=True,
    )

    jobs: Mapped[list["DownloadJob"]] = relationship(back_populates="proxy")

    __table_args__ = (
        # Prevent duplicate proxies
        UniqueConstraint("host", "port", "username", name="uq_proxy_endpoint"),
    )

    @property
    def ytdlp_url(self) -> str:
        """Convert stored format to yt-dlp proxy URL format."""
        return f"http://{self.username}:{self.password}@{self.host}:{self.port}"

    @property
    def display_str(self) -> str:
        """Safe display string (hides password)."""
        return f"{self.host}:{self.port}:{self.username}:***"

    @classmethod
    def from_string(cls, proxy_string: str) -> "Proxy":
        """
        Parse proxy string in format: host:port:username:password
        
        Example: 45.56.180.147:8381:jhrwjggg:axrxuy5m3k18
        """
        parts = proxy_string.strip().split(":")
        if len(parts) != 4:
            raise ValueError(
                f"Invalid proxy format. Expected host:port:username:password, got: {proxy_string!r}"
            )
        host, port_str, username, password = parts
        try:
            port = int(port_str)
        except ValueError:
            raise ValueError(f"Invalid port: {port_str!r}. Must be an integer.")
        if not (1 <= port <= 65535):
            raise ValueError(f"Port {port} out of range (1-65535)")
        return cls(host=host, port=port, username=username, password=password)

    def __repr__(self) -> str:
        return f"<Proxy {self.host}:{self.port} status={self.status}>"
```

### utils/proxy.py

```python
"""
Residential Proxy Pool Manager.

Proxy string format (admin input):
    host:port:username:password
    Example: 45.56.180.147:8381:jhrwjggg:axrxuy5m3k18

yt-dlp format (internal):
    http://username:password@host:port
    Example: http://jhrwjggg:axrxuy5m3k18@45.56.180.147:8381

Rotation strategies:
- round_robin:  Pick proxy with oldest last_used_at (even distribution)
- random:       Pick random active proxy
- least_used:   Pick proxy with lowest total_uses
- least_errors: Pick proxy with lowest total_failures / (total_successes + 1)

Health check:
- Every PROXY__HEALTH_CHECK_INTERVAL seconds (default 300 = 5 minutes)
- Makes GET request to PROXY__HEALTH_CHECK_URL through the proxy
- Expects HTTP 204 (YouTube's generate_204 endpoint)
- Records latency, updates consecutive_failures
- After MAX_CONSECUTIVE_FAILURES: mark DEAD
- After DEAD_PROXY_COOLDOWN: mark ACTIVE again for retry
"""
from __future__ import annotations

import asyncio
import random
import time
from datetime import UTC, datetime, timedelta
from urllib.parse import quote

import httpx
import structlog
from sqlalchemy import select, update

from config.settings import settings
from database.models import Proxy, ProxyStatus
from database.session import get_db
from observability.metrics import (
    proxy_health_check_duration,
    proxy_health_check_total,
    proxy_pool_active_count,
)

log = structlog.get_logger(__name__)


class ProxyPool:
    """
    Manages the proxy pool — health checks, rotation, selection.
    Singleton: instantiated once in bot/main.py, shared across workers via DB.
    
    The pool reads proxy state from DB. This means:
    - Multiple worker processes share the same pool state
    - Health check results are persistent across restarts
    - Admin changes (add/remove/disable) take effect immediately
    """

    def __init__(self) -> None:
        self._lock = asyncio.Lock()
        self._health_check_task: asyncio.Task | None = None

    async def start(self) -> None:
        """Start background health check loop. Call once on startup."""
        if not settings.proxy.enabled:
            log.info("proxy_pool_disabled")
            return
        self._health_check_task = asyncio.create_task(
            self._health_check_loop(), name="proxy-health-check"
        )
        log.info("proxy_pool_started", strategy=settings.proxy.rotation_strategy)

    async def stop(self) -> None:
        """Cancel health check loop. Call on shutdown."""
        if self._health_check_task and not self._health_check_task.done():
            self._health_check_task.cancel()
            try:
                await self._health_check_task
            except asyncio.CancelledError:
                pass

    async def get_proxy_for_url(self, url: str) -> Proxy | None:
        """
        Return the best proxy for this URL, or None if no proxy should be used.
        
        Logic:
        1. If proxy disabled globally → return None
        2. If URL's domain is in NO_PROXY_PLATFORMS → return None  
        3. If URL's domain is in FORCE_PROXY_PLATFORMS → must use proxy
        4. Otherwise → use proxy if available, None if pool empty
        
        Returns None = direct connection (no proxy)
        Returns Proxy = use this proxy's ytdlp_url
        """
        if not settings.proxy.enabled:
            return None

        domain = self._extract_domain(url)
        
        # No-proxy override
        for no_proxy in settings.proxy.no_proxy_platforms:
            if domain.endswith(no_proxy.lstrip(".")):
                log.debug("proxy_skipped_no_proxy_platform", domain=domain)
                return None

        proxy = await self._select_proxy()
        
        # Force-proxy platforms: fail if no proxy available
        for force in settings.proxy.force_proxy_platforms:
            if domain.endswith(force.lstrip(".")):
                if proxy is None:
                    log.warning(
                        "proxy_required_but_pool_empty",
                        domain=domain,
                        hint="Add proxies via /admin proxy add or disable proxy requirement"
                    )
                    # Still return None — let yt-dlp try direct (may fail for YouTube)
                    return None
                break

        return proxy

    async def _select_proxy(self) -> Proxy | None:
        """Select next proxy according to rotation strategy."""
        async with get_db() as session:
            strategy = settings.proxy.rotation_strategy

            if strategy == "round_robin":
                # Pick active proxy with oldest last_used_at
                stmt = (
                    select(Proxy)
                    .where(Proxy.status == ProxyStatus.ACTIVE)
                    .order_by(Proxy.last_used_at.asc().nullsfirst())
                    .limit(1)
                    .with_for_update(skip_locked=True)  # Prevent concurrent workers picking same proxy
                )
            elif strategy == "least_used":
                stmt = (
                    select(Proxy)
                    .where(Proxy.status == ProxyStatus.ACTIVE)
                    .order_by(Proxy.total_uses.asc())
                    .limit(1)
                    .with_for_update(skip_locked=True)
                )
            elif strategy == "least_errors":
                # Order by error rate: failures / (successes + 1)
                # Using raw SQL expression for this calculation
                from sqlalchemy import cast, Float, text
                stmt = (
                    select(Proxy)
                    .where(Proxy.status == ProxyStatus.ACTIVE)
                    .order_by(
                        (Proxy.total_failures / (Proxy.total_successes + 1)).asc()
                    )
                    .limit(1)
                    .with_for_update(skip_locked=True)
                )
            else:  # random
                # Get all active, pick one randomly in Python
                stmt = select(Proxy).where(Proxy.status == ProxyStatus.ACTIVE)
                result = await session.execute(stmt)
                proxies = result.scalars().all()
                if not proxies:
                    return None
                proxy = random.choice(proxies)
                await self._record_proxy_use(session, proxy.id)
                return proxy

            result = await session.execute(stmt)
            proxy = result.scalar_one_or_none()

            if proxy:
                await self._record_proxy_use(session, proxy.id)

            return proxy

    async def _record_proxy_use(self, session, proxy_id) -> None:
        """Update last_used_at and total_uses for selected proxy."""
        await session.execute(
            update(Proxy)
            .where(Proxy.id == proxy_id)
            .values(
                last_used_at=datetime.now(UTC),
                total_uses=Proxy.total_uses + 1,
            )
        )

    async def record_proxy_success(self, proxy_id, latency_ms: float) -> None:
        """Call after successful download through a proxy."""
        async with get_db() as session:
            # Update running average latency
            proxy = await session.get(Proxy, proxy_id)
            if not proxy:
                return
            if proxy.avg_latency_ms is None:
                new_avg = latency_ms
            else:
                # Exponential moving average (alpha=0.3)
                new_avg = 0.3 * latency_ms + 0.7 * proxy.avg_latency_ms
            
            await session.execute(
                update(Proxy)
                .where(Proxy.id == proxy_id)
                .values(
                    total_successes=Proxy.total_successes + 1,
                    consecutive_failures=0,
                    last_success_at=datetime.now(UTC),
                    avg_latency_ms=new_avg,
                )
            )

    async def record_proxy_failure(self, proxy_id) -> None:
        """Call after failed download through a proxy."""
        async with get_db() as session:
            proxy = await session.get(Proxy, proxy_id)
            if not proxy:
                return
            
            new_failures = proxy.consecutive_failures + 1
            new_status = proxy.status
            
            if new_failures >= settings.proxy.max_consecutive_failures:
                new_status = ProxyStatus.DEAD
                log.warning(
                    "proxy_marked_dead",
                    proxy=proxy.display_str,
                    consecutive_failures=new_failures,
                )
            
            await session.execute(
                update(Proxy)
                .where(Proxy.id == proxy_id)
                .values(
                    total_failures=Proxy.total_failures + 1,
                    consecutive_failures=new_failures,
                    status=new_status,
                )
            )
            await proxy_pool_active_count.set(await self._count_active(session))

    async def _count_active(self, session) -> int:
        from sqlalchemy import func, select
        result = await session.execute(
            select(func.count()).select_from(Proxy).where(Proxy.status == ProxyStatus.ACTIVE)
        )
        return result.scalar_one()

    async def _health_check_loop(self) -> None:
        """Background task: periodically health-check all proxies."""
        while True:
            try:
                await asyncio.sleep(settings.proxy.health_check_interval)
                await self._run_health_checks()
            except asyncio.CancelledError:
                break
            except Exception as e:
                log.error("health_check_loop_error", exc_info=True)
                await asyncio.sleep(60)  # Back off on unexpected error

    async def _run_health_checks(self) -> None:
        """Check all proxies concurrently (max 10 at a time)."""
        async with get_db() as session:
            result = await session.execute(select(Proxy))
            proxies = result.scalars().all()

        log.info("running_proxy_health_checks", count=len(proxies))
        
        # Semaphore to limit concurrent health checks
        sem = asyncio.Semaphore(10)
        
        async def check_one(proxy: Proxy) -> None:
            async with sem:
                await self._health_check_single(proxy)

        await asyncio.gather(*[check_one(p) for p in proxies], return_exceptions=True)

    async def _health_check_single(self, proxy: Proxy) -> None:
        """
        Health check one proxy.
        
        - DEAD proxies: check if cooldown expired → try again
        - DISABLED proxies: skip entirely (admin manually disabled)
        - ACTIVE/TESTING proxies: run check
        """
        if proxy.status == ProxyStatus.DISABLED:
            return

        # DEAD proxy: check if cooldown expired
        if proxy.status == ProxyStatus.DEAD:
            if proxy.last_checked_at:
                cooldown_end = proxy.last_checked_at + timedelta(
                    seconds=settings.proxy.dead_proxy_cooldown
                )
                if datetime.now(UTC) < cooldown_end:
                    return  # Still in cooldown, skip

        start = time.monotonic()
        success = False
        
        async with get_db() as session:
            # Mark as TESTING
            await session.execute(
                update(Proxy)
                .where(Proxy.id == proxy.id)
                .values(status=ProxyStatus.TESTING, last_checked_at=datetime.now(UTC))
            )

        with proxy_health_check_duration.labels(proxy=proxy.display_str).time():
            try:
                async with httpx.AsyncClient(
                    proxy=proxy.ytdlp_url,
                    timeout=settings.proxy.health_check_timeout,
                    follow_redirects=True,
                ) as client:
                    resp = await client.get(settings.proxy.health_check_url)
                    # YouTube's generate_204 returns 204 No Content if working
                    # Some proxies return 200 — accept both
                    success = resp.status_code in (200, 204)
            except Exception as e:
                log.debug(
                    "proxy_health_check_failed",
                    proxy=proxy.display_str,
                    error=str(e),
                )
                success = False

        latency_ms = (time.monotonic() - start) * 1000
        proxy_health_check_total.labels(
            proxy=proxy.display_str, result="success" if success else "failure"
        ).inc()

        async with get_db() as session:
            if success:
                proxy_obj = await session.get(Proxy, proxy.id)
                if proxy_obj:
                    new_avg = (
                        0.3 * latency_ms + 0.7 * proxy_obj.avg_latency_ms
                        if proxy_obj.avg_latency_ms
                        else latency_ms
                    )
                    await session.execute(
                        update(Proxy)
                        .where(Proxy.id == proxy.id)
                        .values(
                            status=ProxyStatus.ACTIVE,
                            consecutive_failures=0,
                            last_success_at=datetime.now(UTC),
                            avg_latency_ms=new_avg,
                        )
                    )
                    log.debug(
                        "proxy_health_check_passed",
                        proxy=proxy.display_str,
                        latency_ms=round(latency_ms, 1),
                    )
            else:
                proxy_obj = await session.get(Proxy, proxy.id)
                if proxy_obj:
                    new_failures = proxy_obj.consecutive_failures + 1
                    new_status = (
                        ProxyStatus.DEAD
                        if new_failures >= settings.proxy.max_consecutive_failures
                        else ProxyStatus.ACTIVE
                    )
                    await session.execute(
                        update(Proxy)
                        .where(Proxy.id == proxy.id)
                        .values(
                            status=new_status,
                            consecutive_failures=new_failures,
                            total_failures=Proxy.total_failures + 1,
                        )
                    )
                    if new_status == ProxyStatus.DEAD:
                        log.warning("proxy_marked_dead_by_health_check", proxy=proxy.display_str)

    @staticmethod
    def _extract_domain(url: str) -> str:
        """Extract root domain from URL."""
        from urllib.parse import urlparse
        try:
            return urlparse(url).netloc.lower().lstrip("www.")
        except Exception:
            return ""
```

```python
"""
YouTube Cookie File Manager.

Cookie files allow yt-dlp to authenticate as a logged-in user, enabling:
- Age-restricted video downloads
- Members-only content (YouTube Membership)
- Private videos the bot account has access to
- Region-restricted content (combined with account location)

Cookie file format: Netscape HTTP Cookie File
This is a standard format exported by browser extensions.

How admins upload cookies:
1. Admin runs: /admin cookie upload youtube
2. Bot replies: "Send me the cookie file (.txt)"
3. Admin sends the .txt file as a document
4. Bot validates the file (checks format + makes test request)
5. Bot saves to COOKIES__COOKIES_DIR/youtube/youtube.txt
6. Bot creates/updates CookieFile record in DB

Validation:
- Parse the Netscape format to check it's valid
- Extract earliest expiry date
- Make a test yt-dlp call to verify cookies work
- Report result to admin
"""
from __future__ import annotations

import re
from datetime import UTC, datetime
from pathlib import Path
from typing import Optional

import aiofiles
import structlog

from config.settings import settings
from database.models import CookieFile
from database.session import get_db

log = structlog.get_logger(__name__)


class CookieManager:
    """Manages cookie files for platform authentication."""

    def __init__(self) -> None:
        self._ensure_dirs()

    def _ensure_dirs(self) -> None:
        """Create cookie directories if they don't exist."""
        base = settings.cookies.cookies_dir
        base.mkdir(parents=True, exist_ok=True)
        for platform in settings.cookies.cookie_platforms:
            (base / platform.replace(".", "_")).mkdir(exist_ok=True)

    async def get_cookie_file(self, platform: str) -> Optional[str]:
        """
        Return path to active cookie file for platform, or None.
        
        Only returns a path if:
        1. Cookies globally enabled (COOKIES__ENABLED=true)
        2. Platform is in COOKIES__COOKIE_PLATFORMS
        3. There is an active, valid cookie file for this platform
        4. The file actually exists on disk
        """
        if not settings.cookies.enabled:
            return None

        domain = self._normalize_domain(platform)
        if not any(domain.endswith(p) for p in settings.cookies.cookie_platforms):
            return None

        async with get_db() as session:
            from sqlalchemy import select
            result = await session.execute(
                select(CookieFile)
                .where(
                    CookieFile.platform == self._platform_key(domain),
                    CookieFile.is_active == True,
                )
            )
            cookie_file = result.scalar_one_or_none()

        if not cookie_file:
            return None

        file_path = Path(cookie_file.file_path)
        if not file_path.exists():
            log.warning(
                "cookie_file_missing_on_disk",
                platform=cookie_file.platform,
                path=str(file_path),
            )
            return None

        # Warn if cookies close to expiry
        if cookie_file.expires_at:
            days_left = (cookie_file.expires_at - datetime.now(UTC)).days
            if days_left < 7:
                log.warning(
                    "cookie_file_expiring_soon",
                    platform=cookie_file.platform,
                    days_left=days_left,
                )

        return str(file_path)

    async def save_cookie_file(
        self,
        platform: str,
        content: bytes,
        uploaded_by_user_id=None,
    ) -> tuple[bool, str]:
        """
        Save a cookie file uploaded by admin.
        
        Returns (success, message).
        
        Steps:
        1. Validate Netscape format
        2. Parse expiry dates
        3. Save to disk
        4. Validate cookies work (test yt-dlp call)
        5. Deactivate old cookie files for this platform
        6. Create new CookieFile record
        """
        platform_key = self._platform_key(platform)
        
        # Step 1: Validate format
        try:
            text = content.decode("utf-8")
        except UnicodeDecodeError:
            return False, "❌ File must be UTF-8 encoded text."

        is_valid, parse_error = self._validate_netscape_format(text)
        if not is_valid:
            return False, f"❌ Invalid cookie file format: {parse_error}"

        # Step 2: Parse expiry
        expires_at = self._extract_earliest_expiry(text)

        # Step 3: Save to disk
        cookie_dir = settings.cookies.cookies_dir / platform_key
        cookie_dir.mkdir(parents=True, exist_ok=True)
        
        filename = f"{platform_key}_{datetime.now(UTC).strftime('%Y%m%d_%H%M%S')}.txt"
        file_path = cookie_dir / filename
        
        async with aiofiles.open(file_path, "w", encoding="utf-8") as f:
            await f.write(text)

        log.info("cookie_file_saved", platform=platform_key, path=str(file_path))

        # Step 4: Validate cookies work
        valid, validation_msg = await self._validate_cookies(str(file_path), platform_key)

        # Step 5: Deactivate old files
        async with get_db() as session:
            from sqlalchemy import update
            await session.execute(
                update(CookieFile)
                .where(CookieFile.platform == platform_key)
                .values(is_active=False)
            )

            # Step 6: Create new record
            new_record = CookieFile(
                platform=platform_key,
                filename=filename,
                is_active=True,
                last_validated_at=datetime.now(UTC),
                is_valid=valid,
                validation_error=None if valid else validation_msg,
                uploaded_by_user_id=uploaded_by_user_id,
                expires_at=expires_at,
            )
            session.add(new_record)

        if valid:
            expiry_str = f"\nExpires: {expires_at.strftime('%Y-%m-%d')}" if expires_at else ""
            return True, f"✅ Cookie file uploaded and validated for {platform}.{expiry_str}"
        else:
            return True, (
                f"⚠️ Cookie file saved but validation failed: {validation_msg}\n"
                f"The file is saved. Downloads may still work depending on content type."
            )

    async def _validate_cookies(self, cookie_path: str, platform: str) -> tuple[bool, str]:
        """
        Test cookies by attempting a yt-dlp metadata fetch on a known video.
        
        Uses a short public video — we only need to check if auth works,
        not download anything.
        """
        import asyncio
        
        test_urls = {
            "youtube": "https://www.youtube.com/watch?v=BaW_jenozKc",  # YouTube's test video
        }
        
        test_url = test_urls.get(platform)
        if not test_url:
            # No test URL for this platform — skip validation
            return True, "No test URL configured for this platform"

        try:
            import yt_dlp
            ydl_opts = {
                "quiet": True,
                "no_warnings": True,
                "cookiefile": cookie_path,
                "skip_download": True,
                "extract_flat": False,
            }
            
            # Run in thread (yt-dlp is blocking)
            def _test() -> bool:
                with yt_dlp.YoutubeDL(ydl_opts) as ydl:
                    info = ydl.extract_info(test_url, download=False)
                    return info is not None
            
            result = await asyncio.wait_for(
                asyncio.to_thread(_test),
                timeout=20.0,
            )
            return result, ""
            
        except asyncio.TimeoutError:
            return False, "Validation timed out (20s)"
        except Exception as e:
            error_str = str(e)
            if "Sign in" in error_str or "login" in error_str.lower():
                return False, "Cookies appear to be expired or invalid (login required)"
            return False, f"Validation error: {error_str[:200]}"

    def _validate_netscape_format(self, content: str) -> tuple[bool, str]:
        """
        Validate that file is in Netscape HTTP Cookie File format.
        
        Valid format starts with:
            # Netscape HTTP Cookie File
        
        Each data line has 7 tab-separated fields:
            domain  flag  path  secure  expiry  name  value
        """
        lines = content.strip().splitlines()
        
        if not lines:
            return False, "File is empty"
        
        # Check header (optional but expected)
        has_header = any(
            line.strip().startswith("# Netscape HTTP Cookie File")
            or line.strip().startswith("# HTTP Cookie File")
            for line in lines[:3]
        )
        
        data_lines = [l for l in lines if l.strip() and not l.strip().startswith("#")]
        
        if not data_lines:
            return False, "No cookie data found (only comments)"
        
        # Validate at least first data line
        first = data_lines[0]
        parts = first.split("\t")
        if len(parts) != 7:
            return False, (
                f"Invalid format: expected 7 tab-separated fields per line, "
                f"got {len(parts)} in: {first[:100]!r}"
            )
        
        return True, ""

    def _extract_earliest_expiry(self, content: str) -> Optional[datetime]:
        """Parse cookie file and find earliest expiry date."""
        earliest = None
        for line in content.splitlines():
            if line.startswith("#") or not line.strip():
                continue
            parts = line.split("\t")
            if len(parts) != 7:
                continue
            try:
                expiry_ts = int(parts[4])
                if expiry_ts > 0:  # 0 = session cookie (no expiry)
                    expiry_dt = datetime.fromtimestamp(expiry_ts, tz=UTC)
                    if earliest is None or expiry_dt < earliest:
                        earliest = expiry_dt
            except (ValueError, IndexError):
                continue
        return earliest

    @staticmethod
    def _normalize_domain(platform_or_url: str) -> str:
        """Normalize to domain for matching."""
        if "://" in platform_or_url:
            from urllib.parse import urlparse
            return urlparse(platform_or_url).netloc.lower().lstrip("www.")
        return platform_or_url.lower().lstrip("www.")

    @staticmethod
    def _platform_key(domain: str) -> str:
        """Convert domain to platform key (for file paths and DB)."""
        mapping = {
            "youtube.com": "youtube",
            "youtu.be": "youtube",
            "instagram.com": "instagram",
            "twitter.com": "twitter",
            "x.com": "twitter",
        }
        domain = domain.lower().lstrip("www.")
        return mapping.get(domain, domain.replace(".", "_"))
```

```python
yt-dlp wrapper.

This is the ONLY place in the codebase that calls yt-dlp.

class FormatInfo:
    """A single available format for a video."""

    ext: str

    url: str
    title: str
    duration: Optional[float]      # seconds
    uploader: Optional[str]
    platform: str                   # e.g. "youtube", "instagram"

        """Compute cache key: SHA256(normalized_url + format_quality)."""
        normalized = self._normalize_url(self.url)
        raw = f"{normalized}:{format_quality}"
        self.url_hash = hashlib.sha256(raw.encode()).hexdigest()
        return self.url_hash

    @staticmethod
    def _normalize_url(url: str) -> str:
        """Remove tracking params from URL for better cache hit rate."""
        parsed = urlparse(url)
        # YouTube: keep only 'v' param
        if "youtube.com" in parsed.netloc or "youtu.be" in parsed.netloc:
            from urllib.parse import parse_qs, urlencode
            params = parse_qs(parsed.query)
            if "v" in params:
                clean_query = urlencode({"v": params["v"][0]})
                return f"https://www.youtube.com/watch?{clean_query}"
        return url

def build_format_selector(quality: str, max_size_mb: float = 49.0) -> str:
    """
    Build yt-dlp format selector string based on user preference.
    
    Quality options:
    - "best":  Best quality available (may be very large)
    - "1080":  Up to 1080p
    - "720":   Up to 720p (default)
    - "480":   Up to 480p
    - "360":   Up to 360p
    - "audio": Audio only (best quality audio)
    
    The selector tries to pick the best format within the quality limit.
    Always prefers MP4/M4A for Telegram compatibility.
    """
    if quality == "audio":
        return "bestaudio[ext=m4a]/bestaudio[ext=mp3]/bestaudio"
    
    height_map = {
        "1080": 1080,
        "720": 720,
        "480": 480,
        "360": 360,
    }
    
    if quality in height_map:
        h = height_map[quality]
        return (
            f"bestvideo[height<={h}][ext=mp4]+bestaudio[ext=m4a]"
            f"/bestvideo[height<={h}]+bestaudio"
            f"/best[height<={h}]"
            f"/best"
        )
    
    # "best" — no restriction
    return "bestvideo[ext=mp4]+bestaudio[ext=m4a]/bestvideo+bestaudio/best"


def select_best_format(formats: list[FormatInfo], quality: str) -> Optional[FormatInfo]:
    """
    From available formats, pick the best one under FFMPEG__MAX_SIZE_MB.
    Used to auto-downgrade quality if preferred quality is too large.
    
    Returns None if no suitable format found.
    """
    max_bytes = settings.ffmpeg.max_size_mb * 1024 * 1024
    quality_order = ["1080", "720", "480", "360"]
    
    if quality not in quality_order:
        return None  # "best" or "audio" — can't pre-select
    
    # Start at requested quality, work down
    start_idx = quality_order.index(quality)
    
    for q in quality_order[start_idx:]:
        height = int(q)
        candidates = [
            f for f in formats
            if not f.is_audio_only
            and f.quality_label.rstrip("p").isdigit()
            and int(f.quality_label.rstrip("p")) <= height
            and (f.estimated_size is None or f.estimated_size <= max_bytes)
        ]
        if candidates:
            # Pick highest quality among candidates
            return sorted(
                candidates,
                key=lambda f: (
                    int(f.quality_label.rstrip("p")) if f.quality_label.rstrip("p").isdigit() else 0,
                    f.tbr or 0,
                ),
                reverse=True,
            )[0]
    
    return None

async def fetch_metadata(url: str, user_format_quality: str = "720") -> PreflightResult:
    """
    Fetch video metadata WITHOUT downloading.
    Runs yt-dlp --dump-json equivalent.
    
    Args:
        url: The URL to inspect
        user_format_quality: User's format preference ("720", "1080", "audio", "best")
    
    Returns: PreflightResult with all metadata and format selection
    
    Raises:
        YtDlpExtractError: URL not supported or video unavailable
        YtDlpAuthError: Login required
    """
    proxy = await proxy_pool.get_proxy_for_url(url)
    cookie_file = await cookie_manager.get_cookie_file(url)
    
    log.debug(
        "preflight_started",
        url=url[:80],
        proxy=proxy.display_str if proxy else "none",
        cookies="yes" if cookie_file else "no",
    )
    
    ydl_opts: dict[str, Any] = {
        "quiet": True,
        "no_warnings": True,
        "skip_download": True,
        # Never download during preflight — metadata only
        "listformats": False,
        "socket_timeout": settings.worker.preflight_timeout,
    }
    
    if proxy:
        ydl_opts["proxy"] = proxy.ytdlp_url
    
    if cookie_file:
        ydl_opts["cookiefile"] = cookie_file
    
    def _extract() -> dict:
        with yt_dlp.YoutubeDL(ydl_opts) as ydl:
            return ydl.extract_info(url, download=False)
    
    try:
        info = await asyncio.wait_for(
            asyncio.to_thread(_extract),
            timeout=float(settings.worker.preflight_timeout),
        )
    except asyncio.TimeoutError:
        raise YtDlpExtractError(f"Metadata fetch timed out after {settings.worker.preflight_timeout}s")
    except DownloadError as e:
        error_str = str(e)
        if any(kw in error_str.lower() for kw in ["login", "sign in", "age", "members only", "private"]):
            raise YtDlpAuthError(f"Authentication required: {error_str[:200]}")
        raise YtDlpExtractError(f"Could not extract info: {error_str[:200]}")
    except ExtractorError as e:
        raise YtDlpExtractError(f"Extractor error: {str(e)[:200]}")
    except Exception as e:
        raise YtDlpExtractError(f"Unexpected error: {str(e)[:200]}")
    
    # Parse formats
    raw_formats = info.get("formats", [])
    formats = [
        FormatInfo(
            format_id=f.get("format_id", ""),
            ext=f.get("ext", ""),
            quality_label=f.get("format_note", "") or f"{f.get('height', '')}p",
            filesize=f.get("filesize"),
            filesize_approx=f.get("filesize_approx"),
            vcodec=f.get("vcodec", "none"),
            acodec=f.get("acodec", "none"),
            tbr=f.get("tbr"),
        )
        for f in raw_formats
    ]
    
    # Determine platform from extractor name
    platform = info.get("extractor_key", "unknown").lower()
    
    # Find best format and check size
    best_format = select_best_format(formats, user_format_quality)
    estimated_size_mb = best_format.size_mb if best_format else None
    
    # Use full info filesize if format-level not available
    if estimated_size_mb is None:
        total_filesize = info.get("filesize") or info.get("filesize_approx")
        if total_filesize:
            estimated_size_mb = total_filesize / (1024 * 1024)
    
    large_file_warning = (
        estimated_size_mb is not None
        and estimated_size_mb > settings.ffmpeg.large_file_warn_mb
    )
    
    result = PreflightResult(
        url=url,
        title=info.get("title", "Unknown"),
        duration=info.get("duration"),
        uploader=info.get("uploader"),
        platform=platform,
        thumbnail=info.get("thumbnail"),
        formats=formats,
        selected_format=best_format,
        selected_format_id=build_format_selector(user_format_quality),
        large_file_warning=large_file_warning,
        estimated_size_mb=estimated_size_mb,
    )
    result.compute_url_hash(user_format_quality)
    
    log.info(
        "preflight_complete",
        url=url[:80],
        title=result.title[:50],
        platform=platform,
        formats_count=len(formats),
        estimated_mb=round(estimated_size_mb, 1) if estimated_size_mb else None,
        large_file_warning=large_file_warning,
    )
    
    return result

async def download_media(
    url: str,
    output_dir: Path,
    format_selector: str,
    job_id: str,

    """

        output_dir: Directory to save file (should be /tmp/bot/{job_id}/)
        format_selector: yt-dlp format string from build_format_selector()
        job_id: For logging context
        progress_callback: Async callable called with progress updates
    
    Returns: DownloadResult with file path and metadata
    
    Raises:
        YtDlpDownloadError: Download failed
        YtDlpAuthError: Login required
    """
    proxy = await proxy_pool.get_proxy_for_url(url)
    cookie_file = await cookie_manager.get_cookie_file(url)
    
    output_template = str(output_dir / "%(title)s.%(ext)s")

        "download_started",
        job_id=job_id,
        url=url[:80],
        format=format_selector,
        proxy=proxy.display_str if proxy else "direct",
        cookies="yes" if cookie_file else "no",
    )
    
    # Track progress for throttled updates
    _last_reported_percent = [-1]
    _download_start = [time.monotonic()]
    
    def _progress_hook(d: dict) -> None:
        """Called by yt-dlp on each progress update (blocking context)."""
        if d["status"] == "downloading":
            try:
                downloaded = d.get("downloaded_bytes", 0)
                total = d.get("total_bytes") or d.get("total_bytes_estimate", 0)
                if total > 0:
                    percent = int(downloaded / total * 100)
                    # Only publish every 5% change to avoid Redis spam
                    if percent - _last_reported_percent[0] >= 5:
                        _last_reported_percent[0] = percent
                        speed = d.get("_speed_str", "")
                        eta = d.get("_eta_str", "")
                        # Schedule async callback from sync context
                        asyncio.get_event_loop().call_soon_threadsafe(
                            asyncio.ensure_future,
                            progress_callback(percent, speed, eta),
                        )
            except Exception:
                pass  # Never let progress hook crash the download
    
    ydl_opts: dict[str, Any] = {
        "format": format_selector,
        "outtmpl": output_template,
        "quiet": True,
        "no_warnings": True,
        "progress_hooks": [_progress_hook],
        "socket_timeout": 30,
        "retries": 3,               # yt-dlp internal retries (different from Taskiq retries)
        "fragment_retries": 3,
        "file_access_retries": 3,
        # Merge video+audio automatically
        "merge_output_format": "mp4",
        # Write thumbnail (useful for audio files)
        "writethumbnail": False,
        # Prefer ffmpeg for post-processing
        "prefer_ffmpeg": True,
        # Rate limit to be polite (bytes/second, 0 = unlimited)
        # Disabled by default — proxy already rotates
        "ratelimit": 0,
    }
    
    if proxy:
        ydl_opts["proxy"] = proxy.ytdlp_url
    
    if cookie_file:
        ydl_opts["cookiefile"] = cookie_file
    
    download_start = time.monotonic()
    downloaded_file: Path | None = None
    proxy_failed = False
    
    def _download() -> str:
        """Run download in thread. Returns path of downloaded file."""
        with yt_dlp.YoutubeDL(ydl_opts) as ydl:
            info = ydl.extract_info(url, download=True)
            # Get actual output filename
            filename = ydl.prepare_filename(info)
            # Handle merged output (mp4)
            if not Path(filename).exists():
                # Try with .mp4 extension (after merge)
                filename = str(Path(filename).with_suffix(".mp4"))
            return filename
    
    try:
        filename_str = await asyncio.wait_for(
            asyncio.to_thread(_download),
            timeout=float(settings.worker.job_timeout),
        )
        downloaded_file = Path(filename_str)
        
        if not downloaded_file.exists():
            raise YtDlpDownloadError(f"Download completed but file not found: {filename_str}")
        
    except asyncio.TimeoutError:
        proxy_failed = bool(proxy)
        raise YtDlpDownloadError(
            f"Download timed out after {settings.worker.job_timeout}s"
        )
    except DownloadError as e:
        error_str = str(e)
        proxy_failed = bool(proxy)
        if any(kw in error_str.lower() for kw in ["login", "sign in", "age", "private", "members"]):
            raise YtDlpAuthError(f"Authentication required: {error_str[:300]}")
        raise YtDlpDownloadError(f"Download failed: {error_str[:300]}")
    except Exception as e:
        proxy_failed = bool(proxy)
        raise YtDlpDownloadError(f"Unexpected download error: {str(e)[:300]}")
    finally:
        # Record proxy result
        if proxy:
            if proxy_failed:
                await proxy_pool.record_proxy_failure(proxy.id)
            else:
                download_time = time.monotonic() - download_start
                await proxy_pool.record_proxy_success(proxy.id, download_time * 1000)
    
    download_time = time.monotonic() - download_start
    size_bytes = downloaded_file.stat().st_size
    
    log.info(
        "download_complete",
        job_id=job_id,
        filename=downloaded_file.name,
        size_mb=round(size_bytes / 1024 / 1024, 1),
        duration_s=round(download_time, 1),
        proxy=proxy.display_str if proxy else "direct",
    )
    
    # Extract info for result
    return DownloadResult(
        file_path=downloaded_file,
        filename=downloaded_file.name,
        size_bytes=size_bytes,
        platform=_extract_platform(url),
        title=downloaded_file.stem,
        duration=None,  # Available from preflight result
        proxy_used=proxy.display_str if proxy else None,
        download_time_s=download_time,
    )


def _extract_platform(url: str) -> str:
    """Extract platform name from URL."""
    domain = urlparse(url).netloc.lower().lstrip("www.")
    platform_map = {
        "youtube.com": "youtube",
        "youtu.be": "youtube",
        "instagram.com": "instagram",
        "twitter.com": "twitter",
        "x.com": "twitter",
        "tiktok.com": "tiktok",
        "reddit.com": "reddit",
        "redd.it": "reddit",
        "vimeo.com": "vimeo",
        "twitch.tv": "twitch",
    }
    for domain_key, platform in platform_map.items():
        if domain.endswith(domain_key):
            return platform
    return domain.split(".")[0] if "." in domain else domain
```

### middleware/ssrf.py

```python
"""
SSRF Protection Middleware.

Rejects URLs that resolve to internal/private IP ranges.
Without this, a malicious user could make the bot make requests to:
- Internal services (Redis, PostgreSQL, admin panels)
- Cloud metadata endpoints (169.254.169.254 — AWS/GCP instance metadata)
- Localhost services

This runs FIRST in the middleware chain, before auth or rate limiting.
"""
from __future__ import annotations

import ipaddress
import socket
from typing import Any, Awaitable, Callable

import structlog
from aiogram import BaseMiddleware
from aiogram.types import Message, TelegramObject

log = structlog.get_logger(__name__)

# Private IP ranges that should never be accessed
PRIVATE_RANGES = [
    ipaddress.ip_network("10.0.0.0/8"),
    ipaddress.ip_network("172.16.0.0/12"),
    ipaddress.ip_network("192.168.0.0/16"),
    ipaddress.ip_network("127.0.0.0/8"),
    ipaddress.ip_network("169.254.0.0/16"),  # AWS metadata, link-local
    ipaddress.ip_network("::1/128"),           # IPv6 loopback
    ipaddress.ip_network("fc00::/7"),          # IPv6 private
]


def is_private_ip(ip_str: str) -> bool:
    try:
        addr = ipaddress.ip_address(ip_str)
        return any(addr in network for network in PRIVATE_RANGES)
    except ValueError:
        return False


def is_ssrf_url(url: str) -> bool:
    """
    Returns True if the URL resolves to a private/internal IP.
    Returns False for safe URLs.
    """
    from urllib.parse import urlparse
    try:
        parsed = urlparse(url)
        hostname = parsed.hostname
        if not hostname:
            return False
        
        # Reject IP addresses in private ranges directly
        try:
            addr = ipaddress.ip_address(hostname)
            return is_private_ip(str(addr))
        except ValueError:
            pass  # It's a hostname, not an IP
        
        # Resolve hostname
        resolved = socket.getaddrinfo(hostname, None)
        for _, _, _, _, addr_tuple in resolved:
            ip = addr_tuple[0]
            if is_private_ip(ip):
                return True
        
        return False
    except Exception:
        return False  # If we can't resolve, let yt-dlp handle it


class SSRFProtectionMiddleware(BaseMiddleware):
    """
    Check all URLs in messages against SSRF blocklist.
    Only checks messages that look like URLs.
    """
    
    async def __call__(
        self,
        handler: Callable[[TelegramObject, dict[str, Any]], Awaitable[Any]],
        event: TelegramObject,
        data: dict[str, Any],
    ) -> Any:
        if isinstance(event, Message) and event.text:
            text = event.text.strip()
            # Only check if it looks like a URL
            if text.startswith(("http://", "https://", "ftp://")):
                # Run in thread to avoid blocking event loop (DNS resolution)
                import asyncio
                blocked = await asyncio.to_thread(is_ssrf_url, text)
                if blocked:
                    log.warning(
                        "ssrf_attempt_blocked",
                        url=text[:100],
                        user_id=event.from_user.id if event.from_user else None,
                    )
                    await event.answer(
                        "❌ That URL is not allowed (private/internal address)."
                    )
                    return
        
        return await handler(event, data)
```

```python
"""
Auth Middleware.

Runs on every message:
1. Upsert user (create if new, update last_seen_at)
2. Load user settings (create defaults if not exists)
3. Check if banned → silent drop
4. Inject user object into handler data (available as data["user"])

from __future__ import annotations

from sqlalchemy import select

class AuthMiddleware(BaseMiddleware):

        self,
        handler: Callable[[TelegramObject, dict[str, Any]], Awaitable[Any]],
        event: TelegramObject,
        data: dict[str, Any],

        from_user = None
        if isinstance(event, Message):
            from_user = event.from_user
        elif isinstance(event, CallbackQuery):
            from_user = event.from_user

            return await handler(event, data)

            result = await session.execute(

            )
            user = result.scalar_one_or_none()
            
            if user is None:
                user = User(
                    telegram_id=from_user.id,
                    username=from_user.username,
                    first_name=from_user.first_name,
                )

                await session.flush()  # Get user.id for FK

                return  # Silent drop — banned users get no response
            
            # Inject into handler data
            data["db_user"] = user
            data["user_settings"] = user.settings
        
        return await handler(event, data)
```

### middleware/rate_limit.py

```python
"""
Rate Limit Middleware (Redis Lua atomic token bucket).

Uses a Lua script executed atomically via EVAL.
This is a sliding window counter: each user gets N requests per hour.

Lua script explanation:
1. Get current count for user's rate limit key
2. If count >= limit: return {0, TTL} (not allowed, seconds until reset)
3. INCR count, set EXPIRE if new key
4. Return {1, 0} (allowed)

The atomic EVAL prevents race conditions where two concurrent requests
both read count < limit and both increment past the limit.

Admin rate_limit_override:
- NULL: use global setting
- 0: completely blocked
- N: use N as their personal limit
"""
from __future__ import annotations

from typing import Any, Awaitable, Callable

import structlog
from aiogram import BaseMiddleware
from aiogram.types import TelegramObject, Message, CallbackQuery

from cache.client import get_redis
from config.settings import settings
from observability.metrics import rate_limited_total

log = structlog.get_logger(__name__)

# Lua script: atomic token bucket check+increment
# Returns: [allowed (1/0), seconds_until_reset]
RATE_LIMIT_LUA = """
local key = KEYS[1]
local limit = tonumber(ARGV[1])
local window = tonumber(ARGV[2])

local current = redis.call('GET', key)
if current and tonumber(current) >= limit then
    local ttl = redis.call('TTL', key)
    return {0, ttl}
end

local new_count = redis.call('INCR', key)
if new_count == 1 then
    redis.call('EXPIRE', key, window)
end

return {1, 0}
"""


class RateLimitMiddleware(BaseMiddleware):
    def __init__(self) -> None:
        self._script_sha: str | None = None

    async def _get_script_sha(self) -> str:
        """Load Lua script once, reuse SHA for subsequent EVALSHA calls."""
        if self._script_sha is None:
            redis = await get_redis()
            self._script_sha = await redis.script_load(RATE_LIMIT_LUA)
        return self._script_sha

    async def __call__(
        self,
        handler: Callable[[TelegramObject, dict[str, Any]], Awaitable[Any]],
        event: TelegramObject,
        data: dict[str, Any],
    ) -> Any:
        from_user = None
        if isinstance(event, (Message, CallbackQuery)):
            from_user = event.from_user
        
        if not from_user:
            return await handler(event, data)
        
        # Only rate-limit actual URL downloads, not all commands
        if isinstance(event, Message):
            text = event.text or ""
            is_url = text.strip().startswith(("http://", "https://"))
            is_download_cmd = text.strip().startswith("/download")
            if not (is_url or is_download_cmd):
                return await handler(event, data)
        
        db_user = data.get("db_user")
        
        # Determine limit for this user
        if db_user and db_user.rate_limit_override is not None:
            limit = db_user.rate_limit_override
        else:
            limit = settings.rate_limit.per_hour
        
        # Admin override of 0 = blocked
        if limit == 0:
            if isinstance(event, Message):
                await event.answer("❌ Your account has been restricted.")
            return
        
        redis = await get_redis()
        rate_key = f"ratelimit:{from_user.id}"
        window_seconds = 3600  # 1 hour
        
        try:
            sha = await self._get_script_sha()
            result = await redis.evalsha(sha, 1, rate_key, str(limit), str(window_seconds))
            allowed, reset_in = int(result[0]), int(result[1])
        except Exception as e:
            log.error("rate_limit_check_failed", error=str(e))
            # Fail open: if Redis has an issue, let the request through
            return await handler(event, data)
        
        if not allowed:
            rate_limited_total.inc()
            
            minutes = reset_in // 60
            seconds = reset_in % 60
            reset_str = f"{minutes}m {seconds}s" if minutes else f"{seconds}s"
            
            if isinstance(event, Message):
                await event.answer(
                    f"⏱ Rate limit reached ({limit}/hour).\n"
                    f"Resets in: {reset_str}"
                )
            elif isinstance(event, CallbackQuery):
                await event.answer(f"Rate limit reached. Resets in {reset_str}.", show_alert=True)
            
            # Log to analytics table (async, don't block)
            import asyncio
            if db_user:
                asyncio.create_task(_log_rate_limit(db_user.id))
            
            return
        
        return await handler(event, data)


async def _log_rate_limit(user_id) -> None:
    """Background task to log rate limit event to DB."""
    try:
        from database.models import RateLimitLog
        from database.session import get_db
        async with get_db() as session:
            session.add(RateLimitLog(user_id=user_id))
    except Exception:
        pass  # Analytics failure should never affect request handling
```

### workers/preflight.py

```python
"""
Preflight Worker — fast metadata fetch before actual download.

What it does in ~2 seconds:
1. Check content-aware cache (url_hash in DB) → instant return if cached
2. Fetch metadata via yt-dlp --dump-json (no download)
3. Select best format under 49MB if possible
4. If estimated size > LARGE_FILE_WARN_MB → set flag for user confirmation
5. Cache result in Redis for 1 hour (same URL won't re-fetch)
6. Enqueue download_task (unless large file warning — wait for user response)
"""
from __future__ import annotations

import structlog
from sqlalchemy import select

from config.settings import settings
from database.models import DownloadJob, JobStatus
from database.session import get_db
from queue.broker import broker
from utils.ytdlp import fetch_metadata, YtDlpExtractError, YtDlpAuthError

log = structlog.get_logger(__name__)


@broker.task(
    task_name="preflight",
    max_retries=2,
    retry_delay=10,
    timeout=float(settings.worker.preflight_timeout) + 5,
)
async def preflight_task(
    url: str,
    user_id_str: str,     # UUID as string (JSON serializable)
    job_id_str: str,
    format_quality: str,
    chat_id: int,
    message_id: int,      # The "Queued ✓" message to edit with progress
) -> None:
    """
    Phase 1 of download pipeline: metadata fetch + format selection.
    Runs fast (~2–5s). Chains to download_task on success.
    """
    log_ctx = log.bind(job_id=job_id_str, url=url[:60])
    log_ctx.info("preflight_started")
    
    import uuid
    job_id = uuid.UUID(job_id_str)
    
    # ── 1. Check content-aware cache ──────────────────────────────
    # If this exact URL+format was downloaded before, skip everything
    async with get_db() as session:
        # Compute the hash (need to do it without full preflight for cache check)
        from utils.ytdlp import PreflightResult
        temp = PreflightResult(url=url, title="", duration=None, uploader=None,
                                platform="", thumbnail=None)
        url_hash = temp.compute_url_hash(format_quality)
        
        cached = await session.execute(
            select(DownloadJob).where(
                DownloadJob.url_hash == url_hash,
                DownloadJob.status == JobStatus.DONE,
                DownloadJob.telegram_file_id.isnot(None),
            ).order_by(DownloadJob.completed_at.desc()).limit(1)
        )
        cached_job = cached.scalar_one_or_none()
    
    if cached_job:
        log_ctx.info("cache_hit", cached_job_id=str(cached_job.id))
        await _deliver_cached_result(cached_job, job_id, chat_id, message_id)
        return
    
    # ── 2. Fetch metadata ────────────────────────────────────────
    try:
        preflight = await fetch_metadata(url, format_quality)
    except YtDlpAuthError as e:
        await _fail_job(job_id, str(e), "YtDlpAuthError")
        await _notify_user(chat_id, message_id,
            "❌ This content requires authentication.\n"
            "Ask the admin to upload YouTube cookies (/admin cookie upload youtube)."
        )
        return
    except YtDlpExtractError as e:
        await _fail_job(job_id, str(e), "YtDlpExtractError")
        error_msg = str(e)
        user_msg = "❌ Could not fetch video info."
        if "not available" in error_msg.lower() or "removed" in error_msg.lower():
            user_msg = "❌ This video is unavailable or has been removed."
        elif "private" in error_msg.lower():
            user_msg = "❌ This video is private."
        await _notify_user(chat_id, message_id, user_msg)
        return
    
    # ── 3. Update job with URL hash and platform ──────────────────
    async with get_db() as session:
        job = await session.get(DownloadJob, job_id)
        if job:
            job.url_hash = preflight.url_hash
            job.platform = preflight.platform
    
    # ── 4. Cache preflight result in Redis (1 hour) ────────────────
    from cache.client import get_redis
    redis = await get_redis()
    cache_key = f"preflight:{url_hash}"
    await redis.setex(cache_key, 3600, preflight.title[:200])
    # We store title as proof of cache. Download task has all it needs from args.
    
    # ── 5. Large file warning ──────────────────────────────────────
    if preflight.large_file_warning and preflight.estimated_size_mb:
        size_str = f"{preflight.estimated_size_mb:.0f}MB"
        log_ctx.info("large_file_warning_sent", size_mb=preflight.estimated_size_mb)
        
        # Update job status to PENDING (waiting for user confirmation)
        # The confirmation is handled by a callback query handler
        # Store job_id in Redis so callback can find it
        confirm_key = f"confirm:{chat_id}:{message_id}"
        await redis.setex(confirm_key, 300, job_id_str)  # 5 min to confirm
        
        # Send inline keyboard to user
        from aiogram.types import InlineKeyboardMarkup, InlineKeyboardButton
        keyboard = InlineKeyboardMarkup(inline_keyboard=[[
            InlineKeyboardButton(
                text=f"✅ Yes, download ({size_str})",
                callback_data=f"confirm_download:{job_id_str}",
            ),
            InlineKeyboardButton(
                text="❌ Cancel",
                callback_data=f"cancel_download:{job_id_str}",
            ),
        ]])
        
        await _edit_message(
            chat_id, message_id,
            f"⚠️ <b>Large file warning</b>\n\n"
            f"📹 {preflight.title[:100]}\n"
            f"📦 Estimated size: <b>{size_str}</b>\n\n"
            f"This may take a while and compress to 45MB.\n"
            f"Download anyway?",
            reply_markup=keyboard,
        )
        return  # Wait for user confirmation via callback
    
    # ── 6. Chain to download task ──────────────────────────────────
    log_ctx.info("chaining_to_download_task")
    await _notify_user(
        chat_id, message_id,
        f"⬇️ Downloading...\n📹 {preflight.title[:80]}"
    )
    
    from workers.download import download_task
    await download_task.kiq(
        url=url,
        user_id_str=user_id_str,
        job_id_str=job_id_str,
        format_selector=preflight.selected_format_id,
        format_quality=format_quality,
        chat_id=chat_id,
        message_id=message_id,
    )
```

### workers/download.py

```python
"""
Download Worker — the main pipeline.

Steps:
1. Guard checks (disk space, concurrent jobs)
2. Create temp directory
3. Download via yt-dlp (proxy + cookies applied inside ytdlp.py)
4. Compress if needed (FFmpeg)
5. Upload to Telegram
6. Update DB, notify user, cleanup temp files
"""
from __future__ import annotations

import asyncio
import shutil
import uuid
from datetime import UTC, datetime
from pathlib import Path

import structlog

from config.settings import settings
from database.models import DownloadJob, JobStatus, User
from database.session import get_db
from observability.metrics import (
    jobs_completed_total,
    jobs_failed_total,
    job_duration_seconds,
    bytes_served_total,
)
from queue.broker import broker
from utils.ffmpeg import compress_video, needs_compression, FFmpegError
from utils.quota import check_disk_space, DiskSpaceError, check_and_increment_concurrent, decrement_concurrent
from utils.upload import upload_file
from utils.ytdlp import download_media, YtDlpDownloadError, YtDlpAuthError

log = structlog.get_logger(__name__)


@broker.task(
    task_name="download",
    max_retries=settings.worker.max_retries,
    retry_delay=settings.worker.retry_delay,
    timeout=float(settings.worker.job_timeout),
)
async def download_task(
    url: str,
    user_id_str: str,
    job_id_str: str,
    format_selector: str,
    format_quality: str,
    chat_id: int,
    message_id: int,
) -> None:
    """
    Main download pipeline. Runs in worker process.
    
    This task is retried up to MAX_RETRIES times with RETRY_DELAY seconds between.
    On permanent failure (all retries exhausted): marks job FAILED, notifies user.
    """
    job_id = uuid.UUID(job_id_str)
    user_id = uuid.UUID(user_id_str)
    log_ctx = log.bind(job_id=job_id_str, url=url[:60], format=format_quality)
    
    tmp_dir = settings.disk.tmp_dir / job_id_str
    concurrent_released = False
    start_time = datetime.now(UTC)
    
    try:
        # ── 1. Guard: disk space ──────────────────────────────────
        check_disk_space()  # Raises DiskSpaceError if < MIN_FREE_GB
        
        # ── 2. Guard: concurrent jobs per user ────────────────────
        await check_and_increment_concurrent(user_id_str)
        # From here until finally block, we MUST decrement on exit
        
        # ── 3. Update job status ──────────────────────────────────
        worker_id = f"worker-{asyncio.get_event_loop()}-pid-{__import__('os').getpid()}"
        async with get_db() as session:
            job = await session.get(DownloadJob, job_id)
            if not job:
                log_ctx.error("job_not_found_in_db")
                return
            if job.status == JobStatus.CANCELLED:
                log_ctx.info("job_cancelled_before_start")
                return
            job.status = JobStatus.RUNNING
            job.started_at = start_time
            job.claimed_by = worker_id
            job.heartbeat_at = start_time
        
        # ── 4. Create temp directory ──────────────────────────────
        tmp_dir.mkdir(parents=True, exist_ok=True)
        log_ctx.info("download_task_started", tmp_dir=str(tmp_dir))
        
        # ── 5. Progress callback ─────────────────────────────────
        from cache.progress import publish_progress, start_progress_listener
        
        async def on_progress(percent: int, speed: str, eta: str) -> None:
            await publish_progress(job_id_str, percent, speed, eta)
        
        # Start progress listener (edits Telegram message as updates come in)
        listener_task = asyncio.create_task(
            start_progress_listener(job_id_str, chat_id, message_id)
        )
        
        # ── 6. Download ───────────────────────────────────────────
        try:
            download_result = await download_media(
                url=url,
                output_dir=tmp_dir,
                format_selector=format_selector,
                job_id=job_id_str,
                progress_callback=on_progress,
            )
        finally:
            listener_task.cancel()
            try:
                await listener_task
            except asyncio.CancelledError:
                pass
        
        file_path = download_result.file_path
        log_ctx.info(
            "download_done",
            size_mb=round(download_result.size_bytes / 1024 / 1024, 1),
            download_time_s=round(download_result.download_time_s, 1),
        )
        
        # ── 7. Compression ────────────────────────────────────────
        max_size_bytes = settings.ffmpeg.max_size_mb * 1024 * 1024
        local_api_enabled = settings.local_api.enabled
        
        await _update_message(chat_id, message_id,
            "📦 Processing...\n⏳ Please wait"
        )
        
        if not local_api_enabled and needs_compression(file_path, max_size_bytes):
            log_ctx.info(
                "compression_needed",
                size_mb=round(file_path.stat().st_size / 1024 / 1024, 1),
            )
            try:
                # Update heartbeat before long FFmpeg operation
                await _update_heartbeat(job_id)
                
                file_path = await compress_video(
                    input_path=file_path,
                    target_mb=settings.ffmpeg.target_mb,
                )
                log_ctx.info(
                    "compression_done",
                    compressed_mb=round(file_path.stat().st_size / 1024 / 1024, 1),
                )
            except FFmpegError as e:
                log_ctx.error("compression_failed", error=str(e))
                # Try to send uncompressed anyway (may fail Telegram size check)
                # Better to try than to silently fail
        
        # ── 8. Upload ─────────────────────────────────────────────
        log_ctx.info("uploading_to_telegram")
        await _update_message(chat_id, message_id, "📤 Uploading...")
        
        telegram_file_id = await upload_file(
            chat_id=chat_id,
            file_path=file_path,
            caption=_build_caption(download_result, format_quality),
        )
        
        # ── 9. Update DB ──────────────────────────────────────────
        async with get_db() as session:
            job = await session.get(DownloadJob, job_id)
            if job:
                job.status = JobStatus.DONE
                job.telegram_file_id = telegram_file_id
                job.filename = file_path.name
                job.size_bytes = file_path.stat().st_size
                job.completed_at = datetime.now(UTC)
                job.proxy_id = None  # Will be set by ytdlp.py if proxy used
            
            # Update user counters
            user = await session.get(User, user_id)
            if user:
                user.total_downloads += 1
                user.total_bytes_served += file_path.stat().st_size
        
        # ── 10. Delete the "uploading" message ────────────────────
        try:
            bot = _get_bot()
            await bot.delete_message(chat_id=chat_id, message_id=message_id)
        except Exception:
            pass
        
        # Metrics
        duration = (datetime.now(UTC) - start_time).total_seconds()
        jobs_completed_total.labels(platform=download_result.platform).inc()
        job_duration_seconds.observe(duration)
        bytes_served_total.inc(file_path.stat().st_size)
        
        log_ctx.info(
            "job_complete",
            duration_s=round(duration, 1),
            size_mb=round(file_path.stat().st_size / 1024 / 1024, 1),
        )
    
    except DiskSpaceError as e:
        log_ctx.error("disk_space_error", error=str(e))
        await _fail_and_notify(job_id, str(e), "DiskSpaceError", chat_id, message_id,
            "❌ Server storage full. Please try again later.")
        # Don't retry disk space errors
        raise  # Taskiq will retry but that's ok — disk might free up
    
    except YtDlpAuthError as e:
        log_ctx.error("auth_error", error=str(e))
        await _fail_and_notify(job_id, str(e), "YtDlpAuthError", chat_id, message_id,
            "❌ This content requires authentication (login cookies).\n"
            "Contact admin to configure YouTube cookies.")
        # Don't retry auth errors — they won't resolve themselves
        return
    
    except YtDlpDownloadError as e:
        log_ctx.error("download_error", error=str(e))
        jobs_failed_total.labels(reason="download_error").inc()
        # Re-raise to trigger Taskiq retry
        await _fail_and_notify(job_id, str(e), "YtDlpDownloadError", chat_id, message_id,
            f"❌ Download failed (attempt {_get_retry_count(job_id_str)}/{settings.worker.max_retries}).\n"
            f"Retrying automatically...")
        raise  # Taskiq will retry
    
    except Exception as e:
        log_ctx.error("unexpected_error", exc_info=True)
        jobs_failed_total.labels(reason="unexpected").inc()
        await _fail_and_notify(job_id, str(e), type(e).__name__, chat_id, message_id,
            "❌ An unexpected error occurred. Please try again.")
        raise
    
    finally:
        # ALWAYS release concurrent job slot
        if not concurrent_released:
            await decrement_concurrent(user_id_str)
            concurrent_released = True
        
        # ALWAYS clean up temp files
        if tmp_dir.exists():
            shutil.rmtree(tmp_dir, ignore_errors=True)
            log_ctx.debug("temp_dir_cleaned", path=str(tmp_dir))


def _build_caption(download_result, format_quality: str) -> str:
    size_mb = download_result.size_bytes / 1024 / 1024
    parts = [f"✅ <b>{download_result.title[:80]}</b>"]
    parts.append(f"📦 {size_mb:.1f}MB • {format_quality}")
    if download_result.proxy_used:
        parts.append(f"🔀 Via proxy")
    return "\n".join(parts)


async def _fail_and_notify(job_id, error, error_type, chat_id, message_id, user_msg) -> None:
    async with get_db() as session:
        job = await session.get(DownloadJob, job_id)
        if job and job.status != JobStatus.CANCELLED:
            job.status = JobStatus.FAILED
            job.error_message = error[:1000]
            job.error_type = error_type
    await _update_message(chat_id, message_id, user_msg)


async def _update_message(chat_id, message_id, text) -> None:
    try:
        bot = _get_bot()
        await bot.edit_message_text(
            chat_id=chat_id, message_id=message_id,
            text=text, parse_mode="HTML",
        )
    except Exception:
        pass


async def _update_heartbeat(job_id) -> None:
    async with get_db() as session:
        job = await session.get(DownloadJob, job_id)
        if job:
            job.heartbeat_at = datetime.now(UTC)


def _get_retry_count(job_id_str: str) -> int:
    """Get current retry count from Taskiq context."""
    # Taskiq provides this via task context — simplified here
    return 1


def _get_bot():
    from bot.main import bot_instance
    return bot_instance
```

```python
- Any message that looks like a URL (bare URL)

1. Extract URL from message
2. SSRF check (middleware already ran, but validate URL format)
3. Content-aware cache check (instant delivery if hit)
4. Create DownloadJob record
5. Enqueue preflight_task
6. Reply "✅ Queued"
"""
from __future__ import annotations

import hashlib
import uuid
from urllib.parse import urlparse

from aiogram import Router
from aiogram.filters import Command
from aiogram.types import Message

import structlog
from sqlalchemy import select

from config.settings import settings
from database.models import DownloadJob, JobStatus, UserSettings
from database.session import get_db
from queue.broker import broker
from workers.preflight import preflight_task

log = structlog.get_logger(__name__)
router = Router(name="download")


def extract_url(text: str) -> str | None:
    """Extract URL from message text."""
    text = text.strip()
    # Remove /download prefix
    if text.startswith("/download"):
        text = text[len("/download"):].strip()
    if text.startswith(("http://", "https://")):
        return text.split()[0]  # Take first word (URL only)
    return None


@router.message(Command("download"))
@router.message(lambda m: bool(m.text and m.text.strip().startswith(("http://", "https://"))))
async def handle_download(message: Message, db_user, user_settings: UserSettings) -> None:
    """
    Main download entry point.
    Handler body: 15 lines max. All logic delegated.
    """
    url = extract_url(message.text or "")
    if not url:
        await message.answer(
            "📎 Send me a URL to download.\n"
            "Example: <code>https://youtube.com/watch?v=...</code>",
            parse_mode="HTML",
        )
        return
    
    # Validate URL format
    try:
        parsed = urlparse(url)
        if not parsed.scheme or not parsed.netloc:
            raise ValueError("Invalid URL")
    except Exception:
        await message.answer("❌ Invalid URL format.")
        return
    
    # Create job record
    job_id = uuid.uuid4()
    async with get_db() as session:
        job = DownloadJob(
            id=job_id,
            user_id=db_user.id,
            url=url,
            format_requested=user_settings.format_quality,
            status=JobStatus.PENDING,
        )
        session.add(job)
    
    # Reply immediately (handler must return fast)
    sent = await message.answer(
        f"✅ <b>Queued</b>\n"
        f"🆔 <code>{str(job_id)[:8]}</code>\n"
        f"🎯 Quality: {user_settings.format_quality}",
        parse_mode="HTML",
    )
    
    # Enqueue preflight
    await preflight_task.kiq(
        url=url,
        user_id_str=str(db_user.id),
        job_id_str=str(job_id),
        format_quality=user_settings.format_quality,
        chat_id=message.chat.id,
        message_id=sent.message_id,
    )
    
    log.info("download_enqueued", job_id=str(job_id), url=url[:60])
```

```python
def admin_only(func):
    """Decorator: reject non-admin users silently."""
    import functools
    @functools.wraps(func)
    async def wrapper(message: Message, db_user, *args, **kwargs):
        if not db_user or not db_user.is_admin:
            return  # Silent reject
        return await func(message, db_user, *args, **kwargs)
    return wrapper
```

```python
async def _recover_stale_jobs() -> None:
    """
    On startup, find jobs that were RUNNING when the process crashed.
    Re-enqueue them so users don't wait forever.
    
    A job is "stale" if:
    - status = RUNNING
    - heartbeat_at is older than 5 minutes
    """
    await asyncio.sleep(5)  # Wait for full startup
    
    from datetime import UTC, datetime, timedelta
    from sqlalchemy import select, update
    
    cutoff = datetime.now(UTC) - timedelta(minutes=5)
    
    async with get_db() as session:
        from database.models import DownloadJob, JobStatus
        result = await session.execute(
            select(DownloadJob).where(
                DownloadJob.status == JobStatus.RUNNING,
                DownloadJob.heartbeat_at < cutoff,
            )
        )
        stale_jobs = result.scalars().all()
    
    if not stale_jobs:
        return
    
    log.info("recovering_stale_jobs", count=len(stale_jobs))
    
    for job in stale_jobs:
        log.info("recovering_job", job_id=str(job.id))
        async with get_db() as session:
            j = await session.get(DownloadJob, job.id)
            if j:
                j.status = JobStatus.PENDING
                j.claimed_by = None
                j.retry_count += 1
        
        # Re-enqueue (we don't have chat_id stored in job for notification)
        # Job will be picked up and will try to notify, but message_id is lost
        # In production: store chat_id + message_id in DownloadJob table
        # (add columns: chat_id BigInteger, message_id Integer)

async def webhook_handler(request: web.Request) -> web.Response:
    """Receive Telegram webhook updates."""
    # Verify secret token
    secret = request.headers.get("X-Telegram-Bot-Api-Secret-Token", "")
    if settings.is_production and secret != settings.bot.webhook_secret:
        return web.Response(status=403, text="Forbidden")
    
    bot: Bot = request.app["bot"]
    dp: Dispatcher = request.app["dispatcher"]
    
    data = await request.json()
    from aiogram.types import Update
    update = Update.model_validate(data)
    await dp.feed_update(bot=bot, update=update)
    
    return web.Response(text="OK")


async def health_handler(request: web.Request) -> web.Response:
    """Health check endpoint for Docker HEALTHCHECK and uptime monitors."""
    from cache.client import get_redis
    from database.session import engine
    
    checks: dict[str, str] = {}
    status = 200
    
    # Redis check
    try:
        redis = await get_redis()
        await redis.ping()
        checks["redis"] = "ok"
    except Exception as e:
        checks["redis"] = f"error: {e}"
        status = 503
    
    # DB check
    try:
        async with engine.begin() as conn:
            await conn.execute(__import__("sqlalchemy").text("SELECT 1"))
        checks["database"] = "ok"
    except Exception as e:
        checks["database"] = f"error: {e}"
        status = 503
    
    import json
    return web.Response(
        status=status,
        content_type="application/json",
        text=json.dumps({"status": "ok" if status == 200 else "degraded", "checks": checks}),
    )
```

## PART 15: FORBIDDEN PATTERNS CONTRACT

Every developer on this project signs off on this list. No exceptions.

| ❌ FORBIDDEN | ✅ REPLACEMENT | Why |
|---|---|---|
| `asyncio.Queue` for jobs | Taskiq + Redis broker | In-memory queue loses all jobs on restart |
| `requests` library | `httpx.AsyncClient` or `aiohttp` | Blocking I/O freezes event loop |
| `time.sleep()` | `await asyncio.sleep()` | Blocks entire event loop thread |
| `subprocess.run()` / `Popen()` | `asyncio.create_subprocess_exec()` | Blocking |
| `yt_dlp` called directly in handler | Always `asyncio.to_thread()` in worker | yt-dlp is blocking C extension |
| FFmpeg via `ffmpeg-python` library | Direct `asyncio.create_subprocess_exec()` | Library fights complex args; no HW accel control |
| `os.environ["KEY"]` | `from config.settings import settings` | No validation, crashes at use time not boot |
| Hardcoded proxy strings | DB + admin commands | Can't rotate, can't disable bad proxies |
| Cookie file in code or env var | File on disk, path in DB | Netscape format is multi-line binary data |
| `print()` | `structlog` | Unstructured, no context, unsearchable |
| Bare `except:` | `except SpecificException as e:` | Swallows errors silently |
| yt-dlp without `try/except` | Wrap ALL yt-dlp calls | DownloadError crashes worker |
| String path concatenation | `pathlib.Path` `/` operator | OS-specific separators, no `.exists()` |
| Multiple bot instances | Module-level `bot_instance` singleton | Race conditions, multiple webhook registrations |
| DB session without context manager | `async with get_db() as session:` | Connection leaks |
| Global `async with session:` across await | Fresh session per operation | Connection pool exhaustion |
| Storing proxy password in logs | `proxy.display_str` (masks password) | Credential exposure in logs |
| Cookie file content in DB | File path in DB, content on disk | Cookie files are 10–100KB, not DB data |
| Health check without timeout | Always set `timeout` on httpx | Hangs forever on dead proxy |

## PART 19: UPLOADING YOUTUBE COOKIES — STEP BY STEP

**Step 1: Get the cookies**
1. Install Chrome extension: "Get cookies.txt LOCALLY" 
   (chrome.google.com/webstore — search exact name)
2. Log into youtube.com with your Google account
3. Click the extension icon
4. Click "Export" for youtube.com
5. Save as `youtube.txt`

**Step 2: Upload to bot**
```
/admin cookie upload youtube
```
Bot replies with upload instructions.
Send the `youtube.txt` file as a **Document** (not photo).

**Step 3: Verify**
```
/admin cookie status
```
Expected:
```
✅ youtube
   File: youtube_20260417_120000.txt
   Expires: 2027-04-17
   Last validated: 2026-04-17 12:00
```

**Step 4: Enable in config**
```bash
# .env
COOKIES__ENABLED=true
COOKIES__COOKIE_PLATFORMS=youtube.com,youtu.be
```

**Cookie expiry:**
YouTube cookies typically expire when:
- You log out of the account
- Google detects unusual access patterns
- The account changes password
- After ~1–2 years of non-use

**When cookies expire:**
Bot admin gets a warning in logs. Re-export and re-upload via:
```
/admin cookie upload youtube
```
The new file automatically replaces the old one.
